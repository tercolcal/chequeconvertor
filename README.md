# Cheque Convertor

## Project Purpose
The Cheque Convertor project is designed to facilitate the conversion of handwritten or printed cheque amounts into a digital format that can be processed easily. This makes cheque handling more efficient for businesses and individuals alike, reducing the chances of errors that occur during manual entry.

### Features
- Optical Character Recognition (OCR) to read cheque amounts.
- Ability to export data into various file formats.
- User-friendly interface for easy navigation.

### Getting Started
To get started with the Cheque Convertor, clone the repository and run the necessary scripts to set up the application in your local environment.
